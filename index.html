Campus Compass

A privacy-first indoor navigation tool for VJEC that solves the "lost student" problem. Instead of GPS tracking, it uses a Time-Space Logic Algorithm — predicting faculty locations by combining academic timetables with real-time manual status updates. Navigation without surveillance.

  • Predictive faculty location based on day, time & master schedule
  • Interactive SVG-based schematic campus map
  • Admin dashboard for HODs to override status (leave, class swaps)
  • Search faculty by name or department
  • Fully privacy-centric — no GPS, no hardware beacons
